# PoweredWireWidth視認性確認 / T-083 PoC色切替確認 実機確認記録

忍者、2026-07-16。家老采配（T-097実機確認後の追加依頼）による確認。

## PoweredWireWidth視認性確認（コミット4b703b5、0.45→0.8mm） — OK

- テストモード（Ctrl+T）に入り、a接点(D1、OFF状態)手前の母線区間の通電線（オレンジ）を
  400%ズームでスクリーンショット撮影→ピクセル解析。
- 実測：通電線の太さ約11px(400%時)。コミットメッセージの換算式（96DPI想定1px≈0.26mm）を
  400%ズーム後に割り戻すと約0.72mm相当となり、意図した0.8mmに近い値（目視測定・
  アンチエイリアシングの影響を踏まえ誤差の範囲内）。
- 定性的にも、非通電のグレー線と比べて明確に太く、視認性向上を確認。

## T-083 PoC色切替確認（コミット8a24318、表示メニュートグル） — OK（範囲外の気づきあり）

- 表示(V)メニュー「ダークモード(作図色)(D)」のトグルで、要素配置済みシートの背景・グリッド線・
  接点記号・配線いずれも正しく明暗が切り替わることを確認（Off=白背景黒要素、On=黒背景白要素）。
  Off→On→Offの往復も正常。
- **範囲外の気づき**：シートが0件の状態のみ、トグルのOn/Offに関わらずキャンバスが常に固定の
  濃紺色で表示される（DrawingTheme.Default/Darkのどちらとも異なる色味に見えた）。シート追加後は
  トグル状態に正しく追従する。PoCのスコープ外と見立てるが、シート0件時の背景色の由来は本検証では
  未調査。

## 検証環境の補足

- 両検証ともシート0件からの起動状態で開始したため、まずシート追加が必要だった（T-097検証記録
  参照の既知事象、実害なし）。
