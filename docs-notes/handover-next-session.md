## 3. 試して失敗したアプローチと結果

- 家老が2度、共有git indexの巻き込み事故を起こした（`git add <ファイル> && git commit`で他役の
  ステージ済み変更を巻き込む）。1度目は実害軽微（侍の正当な作業を巻き込んだのみ）、教訓として
  `shared_git_index_commit_sweep.md`メモリへ追記済み。以後は`git commit -m "..." -- <パス>`の
  パス限定形を徹底し再発なし。
