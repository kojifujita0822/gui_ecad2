# T-104増分2(メインツールバー本実装)実機確認記録

忍者作成、2026-07-20。家老委譲、設計書`ecad2-t104-toolbar-tabswitch-design-onmitsu.md`§5「増分3:
検証パイプライン」記載の6観点+追加2観点、計8観点の実機確認結果。ビルド0警告0エラー確認済み。

## 観点別結果

### (1) タブ切替の視認性・クリック操作 → OK

「基本機能」「配置ツール」タブが常時明瞭なラベルで表示され、クリック(`SelectionItemPattern.Select()`)
で正しく内容が切り替わることを確認(基本機能=新規/開く/保存/PDF出力等9ボタン、配置ツール=選択ツール/
a接点配置等の配置系ボタン)。

### (2) 「配置ツール」タブ選択中、基本機能9機能がメニュー/ショートカットで実行可能 → OK

- **テストモード切替(Ctrl+T)**: 「配置ツール」選択中に実行→ステータスバー「モード: Test」に変化、
  再度実行で「モード: Drawing」に復帰。正常動作。
- **行を追加(Ctrl+Shift+Up)**: 実行後スクロールで確認したところ10行→11行に増加。正常動作。
- **行を削除(Ctrl+Shift+Down)**: 実行後11行→10行に復帰。正常動作。
- **新規/開く/上書き保存/PDF出力**: メニューバー(ファイル(F))経由で全項目`IsEnabled=True`を確認
  (実行自体はファイルダイアログ等の副作用を避けるため到達性確認に留めた)。
- **Undo(Ctrl+Z)**: 実行し要素配置操作が正しく取り消されることを確認。
- **Redo(Ctrl+Y)**: 検証中、Undo/Redoボタンが両方`IsEnabled=False`になる場面に遭遇したが、
  これは「シートが1件も無い状態」で発生する既存の無効化条件であり(「基本機能」タブ選択中でも
  同一条件で同じくFalseになることを確認)、**T-104のタブ統合とは無関係な挙動**と切り分けた。
  シート再追加後は正常に有効化された。

### (3) TimerPause対応策 → 対象外(依頼どおり確認不要)

TimerPause機能自体が削除済み(10機能→9機能)につき、対応策の確認自体が不要。

### (4) フロート化操作がCanFloat="False"により実際にできないこと(両タブ) → OK

- 「基本機能」タブ: メニューの「フローティング」`IsEnabled=False`、加えて実際にタブヘッダーを
  350px/400pxドラッグしても新規ウィンドウは生成されず(`EnumWindows`でメインウィンドウ1件のみ)。
  `CanFloat="False"`が確実に機能。
- 「配置ツール」タブ: メニューの「フローティング」`IsEnabled=True`(既知の仕様、T-103のため
  `CanFloat=True`維持と殿裁定済み)。実際にフロート化できることも確認(観点5の前段として実施)。

### (5) ドッキング復帰(Ctrl+Alt+R)で2タブとも正常復元 → OK

「配置ツール」をフロート化(ウィンドウ数2)→`Ctrl+Alt+R`→ウィンドウ数1に復帰、ステータスバー
「パネルレイアウトを既定に戻しました」、`TabItem`数4(基本機能/配置ツール/シート/出力)で正常復元。

### (6) キーボードナビゲーション(Tab順)の破綻がないこと → OK

「基本機能」タブへフォーカス設定→Tab送信1回で、視覚的なフォーカス破線枠が「配置ツール」タブへ
正しく移動(T-104増分1 DoD(4)対策案2の効果が本実装でも維持されていることを確認)。

### (7) 初期選択タブが「配置ツール」になっているか → OK

起動直後、UIA`SelectionItemPattern.IsSelected`で「配置ツール」が`True`、「基本機能」が`False`
であることを確認。

### (8) レイアウト読込失敗メッセージが「レイアウトを既定の状態に更新しました」に変わっているか → OK

意図的にレイアウトファイル(`%AppData%\Ecad2\docking-layout\placement-toolbar.xml`)を旧構成
(1タブのみ)に書き換えて再起動したところ、ステータスバーに**「レイアウトを既定の状態に更新
しました」**と表示され(旧文言「保存済みレイアウトの読込に失敗したため既定で起動しました」から
変更確認)、2タブ構成へ正しく復元された。検証後、書き換えたファイルはバックアップ
(`placement-toolbar.xml.ninja-backup`)から復元済み。

## 総括

依頼された8観点、**全てOK**。T-104増分2(メインツールバー本実装)は完了と判断してよい。
