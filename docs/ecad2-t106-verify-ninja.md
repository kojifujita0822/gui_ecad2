# T-106 配置ツールバーのタブ/インラインバー/ScrollBarダークモード対応 — 実機検証（忍者）

**対象コミット**: `48008f5`（第1弾、タブ文字色・非選択背景）+ `0c3a7d4`（第2弾、ComboBox/未定アイコン/ScrollBar）
**検証日**: 2026-07-21
**手法**: `ecad2-ui-automation`スキル。色検証は目視でなく座標指定の画素採取（支配色集計方式）で理論値と突合。ComboBoxドロップダウンはメインウィンドウとは別のPopupウィンドウのため、EnumWindowsでハンドルを検出後、個別にPrintWindowで撮影した。

## DoD別結果（全項目ライト/ダーク両テーマで実測、理論値と完全一致）
