## P-082：配置系ボタン群のテストモード中無効化 — OK

シートに機器なしの状態でテストモードへ切替え（「ツール(T)」→「テストモード(M)」）、配置系
ボタン6種（a接点/b接点/コイル/端子台/自由線(横線)/接続点）のUIA `IsEnabled`を確認。

| ボタン | IsEnabled |
|---|---|
| a接点配置 (F5) | False |
| b接点配置 (F6) | False |
| コイル配置 (F7) | False |
| 端子台配置 (F8) | False |
| 自由線(横線)記入 (F9) | False |
| 接続点記入 (F10) | False |

全ボタンFalse。スクリーンショットでも全ボタンのグレーアウトを視覚確認。Style既定値化（8箇所→
1箇所への統合）による回帰なし——**判定OK**。

## P-081：機器表「型式」列のテストモード中編集不可化 — OK

機器（a接点、機器名Z1）を配置し機器表へ反映後、テストモードへ切替え、型式セルへ合成マウス
ダブルクリック（同一座標へ2回連続クリック）を実施。

- **テストモード中**：ダブルクリックしても型式セルは編集用テキストボックス化せず、行選択の
  みが発生（右下プロパティパネルに種別/デバイス名等の表示のみ）
- **対照実験（通常モードへ戻し同一操作）**：同じダブルクリックで型式セルが白い編集用
  テキストボックスに変化——編集モードへ正しく入ることを確認

同一操作・同一セルで、テストモード中のみ編集開始がブロックされることを対照実験で直接確認
——**判定OK**（`BeginningEdit`ガードが意図通り機能）。
