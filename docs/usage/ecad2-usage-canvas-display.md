# キャンバス表示

図面を表示するキャンバス領域の表示機能について説明します。

## ズーム

Ctrlキーを押しながらマウスホイールを回すと、キャンバスの表示倍率を変更できます。1回転につき10%刻みで、25%〜400%の範囲で調整できます。現在の倍率はステータスバーに「ズーム: 100%」のように表示されます。

キーボードでのズーム操作や、倍率を100%へ戻すボタンは現時点ではありません。

## グリッド表示の切り替え

「表示」メニューの「グリッド表示」、またはCtrl+Gで、グリッド線の表示・非表示を切り替えられます。非表示にしてもグリッド線が消えるだけで、配置済みの要素や母線などはそのまま表示されます。

この設定はアプリを再起動すると初期状態（表示）に戻ります。

## プロジェクトが無いときの画面

シートが1枚もない状態では、キャンバス領域が濃紺色（ダークモードでは別の暗い色）で表示されます。「ファイル」→「新規」でプロジェクトを作成すると、通常の作業画面（白背景、ダークモードでは暗い背景）に切り替わります。

## 選択中セルの表示

選択しているセルはオレンジ色の枠線で表示されます。線・配線分断・接続点などを選択している場合も、同系統の色でハイライトされます。

## コメント欄・ラベル編集ボックスの閉じ方

行コメント欄（右母線の右側をダブルクリック、またはF2キーで表示）やグループ枠のラベル編集ボックスは、**Enterキーまたはタブキーで確定、Escキーで取り消し**という操作で閉じます。ウィンドウ内の他の場所をクリックしても閉じない仕様になっているため、ご注意ください（別のアプリへ切り替えるなど、ウィンドウ全体が非アクティブになった場合は自動的に確定されます）。
