# 機器表・BOM

右側パネルの「機器表」には、図面に配置した機器（デバイス）の一覧が表示されます。

## 機器の登録・削除は自動

機器表への登録・削除は自動で行われ、手動で行と追加・削除する必要はありません。

- 要素を配置する際、または要素の機器名を入力すると、その機器名が機器表へ自動的に登録されます（同じ機器名が既にある場合は登録済みのものが使われます）。
- 図面上でその機器名を使う要素が1つも無くなると、機器表からも自動的に削除されます。

機器名の自動採番（連番を自動で振る機能）はありません。機器名は自分で入力してください。

## 機器表の並び順

機器名の五十音・アルファベット順（大文字小文字を区別しない）で並びます。配置した順番ではありません。

## 機器表で編集できる項目

機器表の列のうち、**型式列のみ**その場で直接編集できます。機器名・種別の列は表示のみで、機器表から直接編集することはできません（機器名を変更したい場合は、キャンバス上でその要素を選択し、プロパティパネルから変更してください）。

種別（機器の分類）は、その機器名が最初に登録されたときの要素の種類から自動的に決まります。後から要素の種類を変えても、種別表示は追従して変わりません。

## コメント（注記）を付ける

機器にはコメント（注記）を付けることができます。機器表そのものには入力欄がないため、キャンバス上でその機器の要素（接点・コイルなど）を選択し、プロパティパネルからコメントを入力してください。

コメントは機器名単位で共有されます。同じ機器名を持つ複数の要素があっても、コメントは1つだけで、どの要素からも同じ内容が見えます。また、ラダー図本体にも機器名の下に緑色の小さな文字で表示されます。

## PDF出力への反映

PDF出力（メニュー「ファイル」→「PDF出力」、またはCtrl+P）では、機器表の内容がBOM（部品表）ページとクロスリファレンス（相互参照）ページの両方に反映されます。BOM表は画面の機器表と同じ並び順・同じ種別表示で出力され、クロスリファレンス表にはコメントの内容も表示されます。

型式・メーカー・数量のうち、現在編集欄があるのは型式のみです。メーカー・数量は表示のみで、入力する手段は現時点ではありません。
