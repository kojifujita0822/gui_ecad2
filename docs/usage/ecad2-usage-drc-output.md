# 設計チェック(DRC)・出力パネル・検索

図面の設計チェック機能と、画面下部の出力パネル、検索・置換機能について説明します。

## 設計チェックの実行

「ツール」メニューの「設計チェック実行」を選ぶと、図面全体の設計チェック（DRC）が実行されます。実行するたびに、それまでの結果は消え全項目が再チェックされます。

チェックされる主な内容：

- 接点はあるのに駆動するコイルが図面に無い（駆動元不明）
- コイルはあるのに接点が図面に一つも無い（死にリレー）
- 1つのコイルに対して接点が多すぎる（物理的な上限超過）
- 同じ機器名で接点の種類が矛盾している
- 縦の配線が横配線と接続点無しで交差している（電気的に繋がっていない）
- 負荷（コイルなど）が母線まで届いていない
- コイルが2つ以上直列に接続されている（二重コイル）
- 自作パーツの参照が見つからず、仮のa接点として扱われている

## 出力パネルの見方

画面下部の出力パネルに、チェック結果が一覧表示されます。重大度は色分けされます（エラー＝赤、警告＝橙、情報＝グレー）。

行をクリックすると、該当するシート・セルへ自動的にジャンプします。

## 検索・置換

Ctrl+Fで検索バーが表示されます。機器名の完全一致で検索でき、「前へ」「次へ」で該当箇所を移動できます。置換欄に新しい機器名を入力して「置換」（1件のみ）または「全置換」（すべて）を実行できます。

**検索バーを開いている間は、出力パネルの表示内容がDRC結果から検索結果へ切り替わります。** DRC結果を見ながら同時に検索するといった使い方はできません。検索バーを閉じると、出力パネルは元のDRC結果表示に戻ります。
