# メニュー・ツールバー全体構成

画面上部のメニューバーとツールバーの構成を説明します。

## ツールバー

ツールバーは「基本機能」と「配置ツール」の2つのタブに分かれています。タブの見出しをクリックして切り替えます。

### 基本機能タブ

新規作成・開く・上書き保存・元に戻す・やり直し・PDF出力・行を追加(Ctrl+Shift+↑)・行を削除(Ctrl+Shift+↓)・テストモード切替のボタンがあります。

### 配置ツールタブ

選択ツール(Esc)・a接点(F5)・OR a接点(Shift+F5)・b接点(F6)・OR b接点(Shift+F6)・コイル(F7)・端子台(F8)・自由線横(F9)・自由線縦(Shift+F9)・縦分岐線(Shift+F9)・接続点(F10)・配線分断(F10)・グループ枠・自作パーツ(F11)の各ボタンがあります。

Shift+F9・F10は、開いているシートが主回路か制御回路かによって実行される操作が変わります（シート/ドキュメント管理の項目を参照）。

## 今どのツールを使っているか一目でわかる

配置ツールタブのボタンは、現在選択中のツールが背景色と枠線で常時ハイライト表示されます。マウスカーソルを離してもハイライトは維持されるので、今どの操作モードにいるか一目で分かります（接続点記入・配線分断記入は、押した瞬間に処理が完了する操作のため対象外です）。

ボタンを押した瞬間には、これとは別に一時的な押下フィードバック（色の変化）も表示されます。

無効な状態のボタンは薄く表示されます（例：プロジェクトが無い状態での配置系ボタン）。

## パネルの配置は自由に変更できる

ツールバー・左側のパレット・右側のパネル・下部の出力パネルは、いずれもドラッグでの移動・フロート表示（ウィンドウとして独立させる）が可能です。配置を変更した後、「表示」メニューの「現在のレイアウトを既定として保存」を実行すると、次回起動時も同じ配置が復元されます。
