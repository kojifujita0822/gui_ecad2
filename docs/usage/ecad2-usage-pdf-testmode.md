# PDF出力・テストモード

図面をPDFへ出力する方法と、通電状態を確認するテストモードの使い方を説明します。

## PDF出力

「ファイル」メニューの「PDF出力」（Ctrl+P）を実行すると、プレビュー画面が開きます。

- ページ送り：「◀」「▶」ボタン
- 表示倍率の変更：「－」「＋」ボタン、「幅に合わせる」ボタン
- 出力の実行：「PDF出力」ボタンを押し、保存先を選択してください
- プレビューを閉じるだけ（何も出力しない）：「閉じる」ボタン、またはEnterキー

出力範囲は常に図面全体（全シート）で、現在開いているシートだけを出力する選択肢はありません。出力には図面本体のページに加え、クロスリファレンス（相互参照）ページと部品表（BOM）ページも含まれます。

## テストモード

「ツール」メニューの「テストモード」（Ctrl+T）、またはツールバーのテストモードボタンで、通電シミュレーションのON/OFFを切り替えられます。

テストモードに入ると、キャンバス上の接点やスイッチをクリックして動作を確認できます。

| 機器 | 操作方法 |
|---|---|
| 押しボタン | クリックしている間だけON（離すとOFF） |
| セレクトスイッチ | クリックのたびに位置が順番に切り替わる |
| a接点・b接点 | クリックでON/OFFが切り替わる |
| コイル・ランプ・端子台など | クリックしても反応しません（表示のみ） |

通電している配線・機器はオレンジ色でハイライト表示され、通電していない部分はグレーで表示されます。

接点を右クリックすると、「手動でON（強制閉路）」「手動を解除」を選べます。強制ON中の接点は青系の色で区別されます。

タイマー機器は実際の経過時間に応じて自動的に動作します（専用の操作パネルはありません）。

### テストモード中はできないこと

テストモード中は図面の編集（要素の配置・削除、行の追加・削除、元に戻す・やり直しなど）ができません。編集を行うには、テストモードをOFFに戻してください。テストモードをOFFにすると、それまでの通電状態はリセットされます。
