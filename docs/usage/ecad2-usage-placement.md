# 配置操作

ラダー図に接点・コイル・端子台などの要素を配置する方法を説明します。

## 要素を配置する3つの方法

### 1. キーボードで配置する（最も速い方法）

配置したいセルを選択してから、以下のキーを押すと、その場で配置確認欄（配置バー）が表示されます。

| キー | 配置される要素 |
|---|---|
| F5 | a接点 |
| F6 | b接点 |
| F7 | コイル |
| F8 | 端子台 |

### 2. ツールバーボタンから配置する

ツールバーの配置ボタンを押すと配置モードになります。その後、キャンバス上で配置したいセルをクリックすると配置確認欄が表示されます。

### 3. 部品選択パレットから配置する

「自作パーツ」ボタンを押すと、右側のパネルが部品選択リストに切り替わります。組み込みの部品（a接点・b接点・コイル・端子台など）と自分で登録した自作パーツが同じ一覧に表示されるので、リストから配置したい部品をクリックしてください。以降の操作（配置確認欄の表示・確定）は他の方法と共通です。

## 並列回路を追加する（OR配置）

a接点・b接点は、`Shift+F5`（a接点）または`Shift+F6`（b接点）で、既存の回路に対して並列（OR）の接点を追加できます。部品選択パレットの一覧からも、ORバッジの付いたサムネイルを選ぶことで同様に配置できます。

配置位置より上にある回路と自動的に接続され、必要に応じて縦方向の接続線も自動的に引かれます。

## 配置確認欄（配置バー）の操作

要素の配置位置を選ぶと、画面下に配置確認欄が表示されます。

- **部品の変更**：配置確認欄内のドロップダウンから、配置する部品を選び直せます。
- **機器名の入力**：配置確認欄内の欄に機器名を入力できます。
- **確定**：Enterキー、または確定ボタンで配置が確定します。
- **取り消し**：Escキーで配置確認欄を閉じると、何も配置されずに元の状態に戻ります。
- 配置確認欄が表示されている間は、他のショートカットキーは一時的に使えなくなります。
- 配置を確定した後も、選択中のセルと配置モードはそのまま維持されるので、続けて次の要素を配置できます。

## 配置できない場合のメッセージ

| 状況 | 表示されるメッセージ |
|---|---|
| セルを選択せずに配置しようとした | 「配置するセルを先に選択してください」 |
| グリッドの範囲外を選択している | 「選択したセルはグリッド範囲外です」 |
| 既に要素があるセルを選択している | エラー表示（配置できません） |
| プロジェクトを作成していない | 「シートがありません。新規作成（Ctrl+N）から始めてください」 |

## 配置操作と結線操作の違い

a接点・b接点・コイル・端子台の配置は、主回路シート・制御回路シートのどちらでも行えます。

一方、自由線・縦コネクタ・接続点・配線分断などの結線操作（詳しくは「結線操作」の項目を参照）は、シートの種類によって使える・使えないが決まっています。この違いは操作時に意識しておくとよい点です。
