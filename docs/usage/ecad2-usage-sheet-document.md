# シート/ドキュメント管理

図面（プロジェクト）とシートの作成・管理方法を説明します。

## 起動直後とプロジェクトの新規作成

アプリを起動した直後はシートが0枚の状態で、キャンバスは濃紺色の空の状態が表示されます。

「ファイル」→「新規」（Ctrl+N）を実行すると、「シート1」という名前の制御回路シート（行数10・列数20）が1枚作成され、編集を開始できます。

シートが1枚も無い間は、保存系のメニュー・ボタンや配置操作（F5〜F8など）は使用できません。

## シートの追加・削除・改名

- **追加**：左パネルの「シート追加」ボタンから、シート名と種別（主回路／制御回路）を指定して追加します。名前を空欄のままでも自動的に「シートN」という名前が付きます。
- **削除**：左パネルの「シート削除」ボタンから削除します。確認ダイアログは表示されないのでご注意ください。**最後の1枚は削除できません**（図面が0枚になる状態は作れない仕組みです）。シートを削除してページ番号に欠番ができる場合は、その旨の案内メッセージが表示されます（番号を詰め直すことはしません）。
- **改名**：左パネルの「シート名変更」ボタンから変更できます。

## シート種別（主回路／制御回路）

各シートは「主回路」と「制御回路」のどちらかに設定されています。種別によって使える結線操作が異なります。

| 操作 | ショートカット | 使えるシート |
|---|---|---|
| 自由線(横線)記入 | F9 | 主回路のみ |
| 自由線(縦線)記入 | Shift+F9 | 主回路のみ |
| 接続点記入 | F10 | 主回路のみ |
| 縦分岐線記入 | Shift+F9 | 制御回路のみ |
| 配線分断記入 | F10 | 制御回路のみ |

同じショートカットキー（Shift+F9、F10）でも、開いているシートの種別によって実行される操作が変わる点にご注意ください。使えない操作のボタンはグレーアウト表示されます。

なお、a接点・b接点・コイル・端子台などの要素配置（F5〜F8）は、シート種別を問わずどちらでも使えます（配置操作の項目を参照）。

## 新規・開く・保存

- **新規**（Ctrl+N）：未保存の変更がある場合は保存するか確認してから、新しいプロジェクトを開始します。
- **開く**（Ctrl+O）：`.gcad`ファイルを選択して開きます。
- **上書き保存**（Ctrl+S）：現在のファイルへ上書き保存します。まだファイル名が決まっていない場合は「名前を付けて保存」と同じ動作になります。
- **名前を付けて保存**：保存先を毎回選び直して保存します。
- ウィンドウを閉じる際も、未保存の変更があれば確認が入ります。

## シート設定（行数・母線名）

左パネルの「シート設定」ボタンから、行数と母線名（左側・右側）を変更できます。

- 行数は1〜60の範囲で指定できます（新規シート作成時の初期値は10）。
- 行数を減らす場合、削除される行に要素や配線などが残っていると、エラーメッセージが表示され変更できません。先に該当行の内容を削除してから行数を変更してください。
- 母線名は自由なテキストで入力できます（既定値は左「N24」・右「P24」）。空欄も可能です。
