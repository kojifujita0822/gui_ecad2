# ステータスバー・モード表示

画面下部のステータスバーには、現在の操作状態を示す情報が表示されます。

## 表示される項目

| 項目 | 表示例 | 説明 |
|---|---|---|
| ツール | `ツール: Select` | 現在選択中の操作モード（英語表記のまま表示されます。各値の意味は下記「ツール表示の見方」参照） |
| 選択セル | `選択セル: 行3/列5` | 現在選択しているセルの位置（行は1から、列は0から数えます） |
| 操作対象端点 | `操作対象: 始点` | 線（配線・自由線）の端点を選んでいるときのみ表示され、始点か終点かを示します |
| ズーム | `ズーム: 100%` | 現在の表示倍率 |
| 案内/警告メッセージ | 赤字のメッセージ | 操作の案内や、エラー・警告メッセージ |

## ツール表示の見方

「ツール: 」に続く値は、次の8種類のいずれかです。

| 表示値 | 意味 |
|---|---|
| `Select` | 選択モード（要素や配線などを選んで操作する通常の状態） |
| `PlaceElement` | 要素配置モード（a接点・b接点・コイル・端子台など。詳しくは「配置操作」の項目を参照） |
| `PlaceConnector` | 縦コネクタ配置モード（詳しくは「結線操作」の項目を参照） |
| `PlaceFrame` | グループ枠配置モード |
| `PlaceLine` | 自由線配置モード（詳しくは「結線操作」の項目を参照） |
| `PlaceDot` | 接続点配置モード（詳しくは「結線操作」の項目を参照） |
| `PlaceWireBreak` | 配線分断配置モード（詳しくは「結線操作」の項目を参照） |
| `PlaceImage` | 画像配置モード（トレース用の下絵画像を挿入する操作中） |

## 現在のツールを確認する方法

配置ツールバーのボタン（a接点・b接点・コイル・端子台・グループ枠・自作パーツなど）は、現在選択中のものが背景色＋枠線で恒久的にハイライト表示されます。ツールを切り替えるとハイライトも自動的に移動します。

ただし、接続点や配線分断の記入ボタンは押した瞬間に処理が完了する操作のため恒久ハイライトの対象外です。これらのツールを使った後に今どの状態かを確認したい場合は、**ステータスバーの「ツール: 」表示**を確認してください。

なお、開いているシートが主回路シートか制御回路シートかを示す表示はどこにもないため、シート名や操作可能な機能（結線操作の項目を参照）から判断してください。

## 案内・警告メッセージ

ステータスバー右寄りの赤字メッセージには、以下のような案内・エラーが表示されます。

- ツールを選んだ直後の操作案内（例：「配置ツール: a接点 - キャンバスをクリックして配置位置を指定してください」）
- 操作の前提条件が満たされていない場合の案内（例：「シートがありません。新規作成（Ctrl+N）から始めてください」）
- 操作が競合・重複している場合のエラー（例：「この位置には既に配線分断があります」）

メッセージが無い場合は空欄になります。
