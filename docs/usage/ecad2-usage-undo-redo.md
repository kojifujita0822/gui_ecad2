# 元に戻す・やり直し（Undo/Redo）

Ctrl+Z（元に戻す）・Ctrl+Y（やり直し）の使い方と対象範囲を説明します。

## 対象は「シートの追加・削除」のみ

現在、元に戻す・やり直しの対象は**シートの追加・削除のみ**です。以下の操作は対象外で、元に戻すことはできません。

- 要素（接点・コイルなど）の配置・削除
- 結線（自由線・接続点・縦コネクタ・配線分断）の記入・削除・移動
- シートの改名
- シートの行数変更・母線名変更
- 機器名の変更

これらの操作を取り消したい場合は、手動で元に戻す操作を行ってください。

## 使えない場面

以下の状況では、元に戻す・やり直しのボタンがグレーアウトし使用できません。

- **テストモード中**：テストモードは通電状態を観察するための専用モードのため、編集操作全般（元に戻す・やり直しを含む）が無効になります。
- **結線などを記入中（確定前）**：自由線や縦コネクタなどの記入中は、巻き戻し先の前提が崩れることを防ぐため、一時的に無効になります。記入を確定するかEscで取り消してから操作してください。

## 元に戻した後の状態について

シートの追加・削除を元に戻す・やり直しをしても、以下は巻き戻らず現在の状態がそのまま維持されます。

- 選択中のセル（ただし、復元先のシートで行数が足りない場合は末尾行へ自動調整されます）
- 選択中のツール
- 画面のメッセージ表示

## 新規作成・ファイルを開く際は履歴がリセットされる

新規作成やファイルを開く操作を行うと、それまでの元に戻す・やり直しの履歴はクリアされます。別のファイルへ切り替えた後に、以前のファイルの状態へ戻ってしまうことはありません。
