# 結線操作

自由線・接続点・縦コネクタ・配線分断など、要素同士をつなぐ結線操作の使い方を説明します。

## 結線の種類とショートカット

| 操作 | ショートカット | 使えるシート |
|---|---|---|
| 自由線(横線)記入 | F9 | 主回路のみ |
| 自由線(縦線)記入 | Shift+F9 | 主回路のみ |
| 接続点記入 | F10 | 主回路のみ |
| 縦分岐線記入 | Shift+F9 | 制御回路のみ |
| 配線分断記入 | F10 | 制御回路のみ |

シート種別（主回路／制御回路）については「シート/ドキュメント管理」の項目を参照してください。使えない操作のボタンはグレーアウトします。

## 記入の流れ

- **自由線・縦分岐線**：ショートカットキーまたはツールバーボタンで記入を開始すると、矢印キーで線の長さ・方向を調整できます。Enterキーで確定、Escキーで取り消します（何も配置されません）。自由線は水平・垂直方向のみ引けます（斜め線は引けません）。
- **接続点・配線分断**：記入を開始すると、その場ですぐに配置されます（確認欄は表示されません）。同じ場所に配線分断を重ねて記入することはできません。
- シートを切り替えると、記入中の内容は自動的に選択ツールへ戻り、中途半端な状態が残ることはありません。

### 記入中は他の操作ができません

自由線や縦コネクタなどを記入している間（確定前）は、他の配置ショートカット（F5〜F8など）・行の追加/削除・元に戻す/やり直しの操作が一時的に無効になります。誤操作を防ぐための仕様です。記入を確定するかEscで取り消してから、他の操作を行ってください。

## 移動・サイズ変更

配置済みの結線要素はドラッグで移動できます。自由線・縦コネクタは端点をドラッグして長さを変えることもできます（Tabキーで操作対象の端点を切り替え可能）。

## 削除

要素を選択した状態でDeleteキーを押すと削除されます。要素と配線が同じ場所で重なって選択されている場合は、要素（部品）の削除が優先されます。

## 制御回路シートの自動配線

制御回路シートでは、要素を配置するだけで同じ行内の要素同士・母線との横方向の配線が自動的に描画されます。手動での横配線操作は不要です（自動横配線があるため、対応する手動記入メニューもありません）。

一方、主回路シートには自動配線の仕組みが無く、自由線・接続点を使って手動で結線してください。
